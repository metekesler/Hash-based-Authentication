# Hash-based-Authentication
IoT istemci uygulaması, parametre olarak cihaz kimliğini (device ID) ve daha önce sunucu ile paylaşılmış olan gizli anahtarı almaktadır. Geçerli cihaz kimliği ve gizli anahtara sahip cihazlar tarafından oluşturulan hash değeri, istem/yanıt mekanizması kapsamında sunucu tarafından başarıyla doğrulanmaktadır.
